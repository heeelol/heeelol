## 🚀 Featured Projects

### 🎮 [BrainBrawl](https://github.com/heeelol/BrainBrawl) &nbsp;[![Live Demo](https://img.shields.io/badge/Live%20Demo-▶-brightgreen?style=flat-square)](https://brainbrawl-frontend.vercel.app/)
> Real-time 1v1 quiz battle platform with XP progression, custom avatars, and a cosmetic shop

A full-stack gamified learning app built for NUS Orbital (Apollo 11). Features real-time multiplayer via **Socket.IO**, JWT authentication, topic-based quiz pools, a global leaderboard, learning insights dashboard, and E2E testing with **Cypress**.

`React` `Node.js` `Express` `MongoDB` `Socket.IO` `JWT` `TailwindCSS` `Vercel`

---

### 🏎️ [F1 Race Predictor](https://github.com/heeelol/F1-Race-Predictor) &nbsp;[![Demo Video](https://img.shields.io/badge/Demo-YouTube-red?style=flat-square&logo=youtube)](https://youtu.be/qfJKhu7rrgY)
> Full-stack ML web app predicting Formula 1 finishing positions from qualifying data

Trained a **Gradient Boosting Regressor** on 10 years of F1 data (2015–2024) using the FastF1 API. Features a React frontend and Python backend with real-time data fetching, caching, and structured JSON race predictions.

`Python` `scikit-learn` `FastF1` `React` `Vite` `TailwindCSS` `Jupyter`

---

### 🧠 [Rubrix AI](https://github.com/heeelol/Rubrix-AI) &nbsp;[![Hackathon](https://img.shields.io/badge/Hackathon-Semifinalist-gold?style=flat-square)](https://github.com/heeelol/Rubrix-AI)
> Agentic AI learning assistant — SimplifyNext Hackathon Semifinalist (AWS Partnership)

Goes beyond chatbots with true agentic behaviour: **PLAN** (document analysis), **ACT** (custom homework generation), **ADAPT** (continuous difficulty optimisation). Built with React, Node.js, and AWS-integrated LLM backends.

`Python` `Node.js` `React` `TailwindCSS` `AWS` `LLM` `Recharts`

---

### 🤖 [Autonomous Robot — CDE2310](https://github.com/heeelol/CDE2310_G10_2526) &nbsp;[![Docs](https://img.shields.io/badge/Docs-GitHub%20Pages-blue?style=flat-square)](https://kmyming.github.io/CDE2310_G10_2526/)
> ROS2-based autonomous mobile robot with LiDAR SLAM, ArUco vision, and a spring-loaded launcher

Built for a warehouse maze challenge. Implements **frontier-based exploration**, **SLAM Toolbox** mapping, **ArUco marker detection** for delivery zone docking, and a 3D-printed rack-and-pinion launcher. 363 commits. Fully documented with CI/CD via GitHub Actions.

`ROS2` `Python` `C++` `OpenCV` `SLAM` `LiDAR` `Raspberry Pi` `GitHub Actions`

---

### 🔍 [Sift — AI Review Filter](https://github.com/heeelol/Sift)
> LLM-powered spam review detection using NLP + TF-IDF + regex hybrid classification

Scrapes Google Maps reviews via Apify, classifies them into authentic, spam, promotional, and inappropriate categories using a hybrid approach combining open-source LLMs, TF-IDF topic modelling, and regex-based fine-tuning.

`Python` `scikit-learn` `LLM` `Pandas` `TF-IDF` `NLP` `Apify`

### 📚 [ModuleSync — CLI Task Manager](https://github.com/heeelol/tp)
> Java CLI application for NUS students to manage modules, tasks, and academic schedules

Team project built for CS2113 with full OOP design, Gradle build automation, Checkstyle enforcement, JUnit testing, text-UI integration tests, and automated CI via **GitHub Actions**. 342 commits.

`Java` `Gradle` `JUnit` `GitHub Actions` `OOP`
